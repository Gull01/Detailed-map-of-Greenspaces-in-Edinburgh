# Detailed Map of Greenspaces in Edinburgh

This project visualizes key geographic features of Edinburgh, Scotland, including green spaces, related roads, and access points. It provides context and detailed information directly on the map.

## Features
- **Greenspace Visualization**: Highlights the parks and green areas across Edinburgh.
- **Infrastructure Context**: Displays related roads and access points.
- **Interactive Popups**: Detailed information is available for each geographic feature through pop-ups.

## Built With
- Python
- Folium
- GeoPandas
